# Expert Committee App

A web-based Proposal Management System built with HTML, CSS & JavaScript.

## Features
- 📋 New Proposal Submission (EIC)
- ✅ HOD Clearance Workflow
- 👥 Expert Committee Review
- 📄 Work Order Generation
- 📊 Dashboard with Stats

## How to Use
Open `expertcommitee.html` in any browser.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
